# Solving Problems in Java (Easy)

This repository contains solutions to basic Java programming problems for beginners. Each problem demonstrates fundamental Java concepts such as variables, loops, conditional statements, input/output, and simple calculations.

## Problems Included

1. **Hello World** – Print "Hello, World!"  
2. **Greet User** – Take the user's name and greet them  
3. **Even or Odd** – Check if a number is even or odd  
4. **Maximum of 3 Numbers** – Find the largest of three numbers  
5. **Simple Calculator** – Add, Subtract, Multiply, or Divide using if-else or switch  
6. **Print Numbers 1 to N** – Print all numbers from 1 to N  
7. **Multiplication Table** – Print the multiplication table of a number  
8. **Factorial of a Number** – Calculate the factorial of a number  
9. **Sum of Numbers 1 to N** – Calculate the sum of all numbers from 1 to N
